# Git push commands

Run these from a terminal where you have GitHub push access configured.

```bash
git clone https://github.com/nattwarden/SwarmSim-Strategy-Autobuyer.git
cd SwarmSim-Strategy-Autobuyer

# Copy the contents of this bootstrap folder into the repo root, then:
git add .
git commit -m "Add SwarmBot 0.7.9 baseline repository structure"
git push origin main
```

If the repo is completely empty and clone warns about an empty repository, that is fine.
