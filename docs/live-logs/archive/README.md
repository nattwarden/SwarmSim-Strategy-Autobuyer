# live-logs/archive

Raw browser-export artifacts from SwarmSim Strategy Autobuyer versions
0.10.0 through 0.12.3.

These files are preserved as forensic evidence for historical version
verifications. They are not active reference material.

## Why they are here

The verification SHA records (`../verification-0.12.2-exact-implementation-sha.*`
and `../verification-0.12.3-exact-implementation-sha.*`) enumerate these
files as the evidence artifacts that existed at the time of each formal
verification run. Their paths in those records reflect the original
`docs/live-logs/` root location; those records have not been rewritten.

If a claim in Books 01–03 needs payload-level verification, open the
specific file here. Do not use these files as starting points for new work.

## What to read instead

All meaningful conclusions from these exports have been distilled into:

- `../books/BOOK-01-base-mechanics-and-claims.md`
- `../books/BOOK-02-energy-house-of-mirrors-and-lab.md`
- `../books/BOOK-03-verification-history-and-artifacts.md`

Current acceptance evidence is in the `../` root as
`live-real-save-0.12.3-*` files.
