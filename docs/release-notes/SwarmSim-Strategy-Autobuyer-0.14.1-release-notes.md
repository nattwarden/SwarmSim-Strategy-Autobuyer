# SwarmSim Strategy Autobuyer 0.14.1

## What changed

- Replaced the confusing default control surface with two prominent operating
  choices: Advisor and Autobuyer.
- Advisor keeps Smart analysis active but disables purchases. Autobuyer permits
  the existing safe, reversible purchase authority.
- Moved legacy presets, detailed strategy tuning, exports, and Laboratory tools
  into one collapsed Advanced settings and tools section. Existing values and
  controls remain available when needed.
- Fixed button colors so text remains readable independently of the game's
  global button styling.
- Made The Swarm Council movable and resizable. Its position and size persist
  between sessions and are included in Reset views.

## Safety and scope

- No strategy scoring, purchase ranking, Laboratory formula, or execution
  adapter changed.
- Advisor and Autobuyer only map to the existing Smart authority flags.
- Ability auto-cast and auto-Ascension safety defaults remain disabled.
