# SwarmSim Strategy Autobuyer 0.12.2 Release Notes

## Summary

0.12.2 is a narrow Laboratory patch for the live House of Mirrors canonical resolver and blocked-action observability.

## What changed

- Fixed canonical House of Mirrors affected-unit resolution to use the runtime `clonearmy.effect[]` compound-unit ids.
- Kept the affected-id set canonical and base-id keyed for Laboratory snapshot and Phase 1 compatibility.
- Added explicit blocked-action observability for energy-blocked Clone Larvae and House of Mirrors captures.
- Added reserve provenance labels for live capture, scenario overrides, and config-derived captures.
- Added 0.12.2 Laboratory verifier/evidence paths.

## What did not change

- Normal strategy behavior remains unchanged.
- Planner ranking remains unchanged.
- Auto-cast defaults remain unchanged.
- Ascend defaults remain unchanged.
- Safety defaults remain unchanged.
- Phase 1 scope remains Laboratory-only and simulation-only.
