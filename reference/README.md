Do not read these files to determine current bot behavior or claim status.
Instead, read the classified claim book:

```text

```

- CONFIRMED
- PARTIALLY CONFIRMED
- UNCONFIRMED
- BOT BOUNDARY
- OUTDATED / WRONG
- OUT OF SCOPE

Open the raw source files only when a specific claim needs payload-level verification
against the original wording.
